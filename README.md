## 🛠️ Instalación y Uso
1. Clonar el repositorio:  
   ```bash
   git clone https://github.com/camilognzz/portfolio.git
   ```
2. Instalar dependencias:  
   ```bash
   npm install
   ```
3. Ejecutar el proyecto:  
   ```bash
   npm run dev
   ```
